jQuery Functions Demo

1. map()

Transforms each element in an array or object and returns a new array of results. 

Syntax: 
$.map(array, function(element, index) { return new_value; });

Use Case: 
Convert an array of numbers to their squares.

2. grep()

Filters an array based on a condition and returns matching elements. 

Syntax: 
$.grep(array, function(element, index) { return condition; }, [invert]);

Use Case: 
Find all even numbers in an array.

3. extend()

Merge two or more objects into the first object.

Syntax: 
$.extend(target, object1, [objectN]);

Use Case:

4. each()

Iterates over an array or object and applies a function to each element. 

Syntax:

Syntax:
