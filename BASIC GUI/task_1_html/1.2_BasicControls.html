Form Input Types
Type Use Demo
Label Defines a label for an input element.
Text To get single-line text.
Password To get a password.
Email Used for entering an email address. Validates email format.
Number Allows the user to enter a number with optional min, max, and step values.
Search Provides a search box for entering search queries.
Checkbox Allows the user to select multiple options from a set. Option1
Option2
Radio Allows the user to select one option from a set of choices. option-1
option-2
Color Provides a color picker to choose a color.
Date Allows the user to select a date.
DateTime-Local Allows the user to select a date and time.
File Allows the user to upload a file.
Hidden Stores data that is not visible to the user.
Image Displays an image as a submit button.
Month Allows the user to select a month and year.
Range Allows the user to select a value from a range.
Tel Used for entering a telephone number.
Time Allows the user to select a time.
Week Allows the user to select a week and year.
Textarea Used for multi-line text input.
Select Creates a dropdown list of options.
Button Triggers a JavaScript function or performs an action.
Submit Submits the form data to the server.
Reset Resets the form to its default values.
